Gestión simple del impuesto Recargo de Equivalencia para WooCommerce.

== Description ==

WooCommerce calcula los impuestos en función de Zonas Geográficas, y por tanto no es posible hacer ninguna configuración para el Recargo de Equivalencia. Con este plugin es posible añadir el Recargo de Equivalencia a los Clientes que lo requieran en base al Rol del Cliente.
Desde la página de administración del plugin se seleccionan los Impuestos y los Roles que están sujetos a Recargo de Equivalencia.

= Características =

* Aplica a Productos Simples y Variables
* Se basa en filtros para determinar cuándo debe aplicarse el Recargo de Equivalencia
* Compatible con WooCommerce 2.* y 3.* 

== Instrucciones de Instalación  ==

* Use la característica de instalación automática de plugines en su panel de administración de WordPress, y busque “WooCommerce Recargo de Equivalencia”. 
* Ir a 'WooCoomerce' > 'Recargo de Equivalencia' para configurar el plugin.

Más información: [Cómo Instalar Plugins](https://codex.wordpress.org/Managing_Plugins).
